库简介

pyinstxtractorcn 是一个功能强大的 Python 库，专门用于解包和反编译由 PyInstaller 工具创建的 Windows 可执行文件 (.exe)。它能提取出打包在内部的 Python 字节码、资源文件等，是进行软件分析、学习和研究的有效工具。

本库提供了配套的图形界面工具 pyinstxtractorcnGUI，两者将同步更新，为用户提供从命令行到图形界面的完整解决方案。

安装与使用

通过 Python 的包管理工具 pip 即可轻松安装：

pip install pyinstxtractorcn

基本使用示例：

import pyinstxtractorcn
# 解包 your_program.exe
pyinstxtractorcn.dcp("your_program.exe", "output_directory")
